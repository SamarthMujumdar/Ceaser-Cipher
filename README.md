# 🔐 Caesar Cipher Cryptography

![Python](https://img.shields.io/badge/Python-3.10%2B-blue.svg)
![Cybersecurity](https://img.shields.io/badge/Focus-Cryptography-red.svg)

**Project Objective:** To implement a programmatic version of the Caesar Cipher, one of the foundational algorithms in cryptography. This Python script provides a secure environment to encode sensitive plaintext strings or decode intercepted ciphertexts by shifting alphanumeric values across a variable key index.

* **Samarth Mujumdar** - Python Implementation

## 🚀 Technical Highlights
* **String Manipulation:** Advanced handling of ASCII values to securely shift characters while maintaining casing and ignoring special symbols.
* **Modular Math Logic:** Utilizes modulo operators to seamlessly wrap shifts around the alphabet (e.g., shifting 'Z' forward wraps back to 'A').
* **Two-Way Processing:** Fully functioning logic for both secure encryption and rapid decryption based on user-defined shift keys.
